# Node.js Random Color Generator

A random color generator CLI app that will be able to generate completely random colors

`````bash
$ node index.js
###############################
###############################
###############################
#####                     #####
#####       #ff0000       #####
#####                     #####
###############################
###############################
###############################
```

Or random colors based on input:

````bash
$ node index.js red light
###############################
###############################
###############################
#####                     #####
#####       #ff0201       #####
#####                     #####
###############################
###############################
###############################
```
`````
